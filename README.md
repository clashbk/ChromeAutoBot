### 功能介绍：

自动化操作chrome内核浏览器，完成日常需要重复的动作。可自定义规则，再也不愁积分不够了，薅羊毛利器。

### 使用教程：

1、点击项目旁边的“+”按钮，新建项目。

2、编写项目规则，保存。

3、点击“刷新”按钮，载入新建的项目。

4、点击项目的“登录”按钮，登录你的账号。

5、点击项目的“测试”按钮，进行测试。

6、如果项目需要自动运行，勾选“启用”。
